# Landing Page — Nicho Academico

![Preview](docs/preview.png)


Este repositório local faz parte do meu portfólio. Esta LP de Academico foi desenvolvida profissionalmente (no contexto do meu trabalho) e mantenho uma versão pública no GitHub para apresentar meu processo e padrão de entrega.

- Repositório público: https://github.com/svlucas7/zap-lp-nicho-academico
- Tipo: Landing Page estática com foco em conversão
- Tom: portfólio profissional (clareza, performance e organização)



## ✨ Visão geral
- Hero com proposta de valor clara e CTA destacado
- Benefícios, prova social e FAQ para reduzir objeções
- Layout responsivo e leve, otimizado para velocidade
- Preparado para integrações de marketing (pixel, GTM, formulários)

## 🛠️ Tecnologias e padrões
- HTML semântico e Tailwind CSS
- JavaScript vanilla para interações pontuais
- Imagens otimizadas e lazy loading quando aplicável
- Deploy via Netlify ou GitHub Pages (opcional)

## 📁 Estrutura
- index.html — página principal
- public/ — assets estáticos
- src/ — fontes (quando aplicável)
- 	ailwind.config.js, postcss.config.js — configuração (quando presente)
- 
etlify.toml  deploy (quando usado)

##  Rodar localmente
Windows PowerShell:
`powershell
Start-Process .\index.html
# ou servidor local para testar lazy loading
python -m http.server 8000
# http://localhost:8000
`
